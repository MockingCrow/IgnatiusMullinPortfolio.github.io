Card Game and Website

Backend Development

A group project based around a card game called “Bannermen”. We created a webpage which displayed rules and a leaderboard of the top 10 players. The website included a downloadable Java app through which the card game could be played digitally via local machine or LAN using a Java GUI. The app includes a login and registration system which connected to a database that stored hashed login information along with user statistics such as wins and losses.

Contribution

  • Used a MySQL database to store usernames, hashed passwords and user statistics
  • Created functions in NetBeans that could add user information, verify login credentials, and update leaderboard information based on real time wins and losses.
  • Worked with front end team to connect the webpage to my backend functions
