Map Coloring Algorithm

C++ Algorithm

Program based on the four-color map theorem. The program takes an input file of a grid of letters where each letter starting from “A” represents a different country and all parts of a country must be contiguous. Above the map of letters in the input file, the user must provide the number of countries, the number of rows in the grid, and the number of columns; given in the format:
[Integer Integer Integer] (see infile.txt in the Github for an example).
Upon receiving this input, the program will assign each country with a color such that no two adjacent countries have the same assigned color. As per the four-color map theorem, the program will use no more than 4 colors to accomplish this task. This program as created using C++ and Visual Studio Code.
