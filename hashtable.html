Hashtable Timing

C++ Algorithm

A hash table that takes in a text file and calculates the number of instances of each word in that text. Supports linear probing, quadratic probing, and double hashing. There is a timing program that calculates the time for the input file to be stored in the hash table. This program also supports timing using a map. This program was created using C++.
